# SME Pulse Dashboard

This project contains the SME Pulse Dashboard, which renders an HTML dashboard embedded inside a Streamlit application.

## Prerequisites

- Python 3.7 or higher

## Setup

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```

## Project Structure

- `app.py`: The main Streamlit application script.
- `index.html`: The HTML file containing the dashboard structure.
- `css/`: Directory containing CSS styles.
- `js/`: Directory containing JavaScript code.
- `data/`: Directory for data files (e.g., `snapshot.json`).
- `requirements.txt`: Python dependencies required to run the project.
